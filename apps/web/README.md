# @politigraph/web

Landing page, admin panel, and document to be rendered as SPA

## Tech Stack

- Nuxt + Nuxt Content
- Tailwind
- Carbon Components
- v-network-graph

## Scripts

- `dev` run server in dev mode
- `build` build web as a static SPA website

## Environment Variables

- `NUXT_PUBLIC_BASE_URL` frontend base URL
