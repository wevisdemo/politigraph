# @politigraph/docs

Landing page and documentations to be built as static website

## Tech Stack

- Astro + Startlight
- Tailwind
- Vue
- v-network-graph

## Scripts

- `dev` run server in dev mode
- `build` build web as a static website

## Environment Variables

- `POLITIGRAPH_URL` for GraphQL endpoint (optional, default to https://politigraph.wevis.info)
